## 🚀 Ready to Deploy

### Quick Start (3 steps):

1. **Install & Build** (5 min)
   ```bash
   npm install
   cd contract
   cargo build --release --target wasm32-unknown-unknown
   ```

2. **Deploy Contract** (10 min)
   ```bash
   stellar contract deploy \
     --wasm contract/target/wasm32-unknown-unknown/release/minidonaa.wasm \
     --source alice \
     --network testnet \
     --alias minidonaa
   ```

3. **Run App** (1 min)
   ```bash
   npm run dev
   # Open http://localhost:3000
   ```

**Total Time: ~1 hour** ⚡
